# Security Policy

## Reporting a vulnerability

Please report security vulnerabilities privately to **security@mosadd.com**.

Do **not** open a public issue for security reports. Include, if you can:

- a description of the issue and its impact,
- steps to reproduce (a minimal example is ideal),
- affected package(s) and version(s).

We'll acknowledge your report as quickly as we can and keep you updated on the
fix. Coordinated disclosure is appreciated — please give us reasonable time to
release a fix before any public write-up.

## Scope

This repository contains client-side detection libraries and examples. There is
no hosted service or secret material in this repository. Reports about the
hosted product at https://mosadd.com are also welcome at the same address.
